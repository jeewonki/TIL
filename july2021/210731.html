2021.07.31

Navigator객체

"Cross-Browsing 문제"
ECMA, W3C에 의해 여러 브라우저들의 동작 방식이 표준화되기 전에는, 넷스케이프와 IE 의 API의 차이 때문에 개발자들의 혼동이 발생했음. 표준화된 이후에도 Specification들을 구현하는 방식이 브라우저마다 서로 다르기 때문에, 브라우저의 정보를 얻어 그에 대응하는 것이 필요함

console.dir(navigator)
:navigator객체의 필드를 확인 가능
console.dir(navigator.appName)
:앱의 이름(크롬과 파이어폭스 등은 Nescape로 표시되기 때문에 실질적으로는 사용x)
console.dir(navigator.appVersion)
:자신의 컴퓨터의 운영체제, 브라우저의 웹 키트, 프라우저 이름 확인 가능
console.dir(navigator.userAgent)
:브라우저에서 서버 측에 전송하는 정보의 내용(보통 appVersion과 동일)
console.dir(navigator.platform)
:win32, Linux x86_64 등 자신의 운영체제 확인 가능
