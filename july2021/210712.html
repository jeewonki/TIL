선택자의 개념 심화

CSS의 선택자
태그 선택자
li{color:red;}
:모든 <li>태그에 적용
아이디 선택자
id 속성값이 들어간 태그에만 적용
한 id 속성값은 하나의 태그만을 가리킴(클래스와 구분할 것) #select{color:red;}
<p id="select"> contents </p>
: contents가 빨갛게 표시
클래스 선택자
class 속성값이 들어간 태그에만 적용
class는 대상을 관리하기 쉽도록 그룹 지정한 것
따라서, id와 달리 여러 태그를 가리킬 수 있음
<span class="css">li{color:red;}</span>
.css{ font-family:'Ubuntu Mono', monospace; color:#39FF74;background-color:#000000;}
: 이 사이트에 적용된 클래스의 예
