Object Model

javaScript는 브라우저를 제어하기 위해 사용
제어 가능한 각각의 요소들을 "객체"의 형태로 제공
html 문서의 객체화는, 각각의 브라우저에서 이루어짐 documents.getElementsByTagName("tag"): 태그의 객체를 반환
태그는 복수이므로, array의 형태로 반환

DOM/BOM/JS Core

1. DOM(Document Object Model): 웹 페이지의 문서를 제어
2. BOM(Browser Object Model): 브라우저의 요소를 제어
3. Core: 자바스크립트에서 기본 제공되는 메소드
모든 메소드들은, 전역 객체인 window에 포함되어 있음
