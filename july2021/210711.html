텍스트 관련 CSS 속성

font-size
크기 단위
px
px는 웹에서 가장 기본적인 크기 단위
화면을 구성하는 1개의 점이 1px
rem(root em)
최상의 html 태그의 기본값의 배수
html 태그의 폰트 크기가 스타일 시트에서 바뀌면, 이것도 바뀜
em
부모 태그의 폰트 크기의 배수
ex) body 태그 안쪽에 h1과 p태그가 있다
h1과 p는 body의 자식 태그이므로, body 태그의 폰트 크기의 몇 배
vw, vh
뷰포트 관련 폰트 크기로, vw는 화면 가로 크기의, vh는 화면 세로 크기의 1/100
브라우저 창의 크기에 따라 폰트 크기가 달라짐
Color

p{color:green;}

색상을 표현하는 세 가지 방법

  1. 키워드: 색상의 영문 이름을 값의 키워드로 사용
  2. RGB: rgb(r, g, b)
  3. HEX(#RRGGBB)

    여섯 자리의 16진수
    두 자리씩 R, G, B를 나타냄 ex) #F5DF4D=rgb(245,223,77)

font-family
font-family:[글꼴], [글꼴유형];
글꼴유형은 generic family로, serif, sans-serif처럼 글꼴의 큰 분류를 나타냄
Font-weight
굵기 단위
100~900: 폰트마다 모두 있을 수도 있고, 없을 수도 있음
normal, bold, bolder, lighter
Line-height
글자의 줄 높이
글자 크기의 배수, 또는 고정 단위(px) 가독성을 위해, 고정 단위는 폰트 사이즈보다 크게 지정할 것!
