CSS의 캐스케이딩

웹 페이지에 대한 결정권이 있는 요소들:
브라우저, 사용자, 저자
이들이 동시에 하나의 페이지를 수정하려고 할 때, 누구를 우선순위로 두어야 하는가?

우선순위: 웹브라우저<사용자<저자

하나의 태그에 여러 CSS 스타일이 적용되었을 때의 우선순위:
style속성>id선택자>class선택자>태그 선택자

!important키워드를 붙이면, 우선순위가 가장 높아짐

반응형 디자인/미디어 쿼리

@media(min-width:800px){
div{display:none}
}
