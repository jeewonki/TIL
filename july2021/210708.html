오픈소스아카데미 WEB 과정 HTML+CSS(수코딩)

텍스트 강조 태그
<strong>: 중요한 텍스트 <em>: 강조

둘의 차이는, 의미상 "반드시 전달해야 하는가?"의 의미가 있는지의 차이이다.

<b>: Bold <i>: Italic

이 둘은 외형만 바꿀 뿐, 의미는 없음

목록 태그

<ul>: Unordered list
<ol>: Ordered list,
type 속성으로 번호의 종류를 바꿀 수 있음
<li>: list item

<dl>: Definition list
<dt>: Definition title
<dd>: Definition data
제목과 그에 대한 설명이 있을 때 사용

CSS 기본
내부 스타일 시트: <style> 사이에 css 스타일 입력
외부 스타일 시트: 별도의 css 파일 만듬
<link href="경로" rel="stylesheet"> 인라인: 모든 태그에 style 속성 추가
