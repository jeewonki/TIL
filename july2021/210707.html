프론트엔드와 백엔드의 개념

프론트엔드: 사용자에게 보여지는 영역
사용 언어: HTML, CSS, JS
vue.js, react.js, angular.js

백엔드: 사용자에게 보이지 않는 영역, 데이터를 가공
사용 언어: JAVA, php, python

"백엔드 개발을 위해서는 프론트엔드도 필수적으로 알아야 한다"
Js의 DOM 까지만 배우고, 백엔드로 넘어갈 것

태그
< 와 > 사이에 태그 이름
보여지기 위한 꺾쇠는 &lt;, &gt;로 나타냄
</> 태그로 닫을 수 있는 태그와, 닫을 수 없는 태그(빈 태그)
태그의 속성: 태그를 의미나 기능적으로 보조
ex) lang="en": <html> 태그에 '사용 언어'라는 정보를 추가
기본 구조
  • <!DOCTYPE html> : 최신 버전의 html 사용
  • <html> : html 문서의 시작을 알림
  • <head> : 문서의 정보 표현
  • <meta> : <head>에서 인코딩과 뷰포트 등을 결정
  • <title> : <head>문서의 제목 표현(탭 이름으로 표기)
  • <body> : 브라우저에 표현되는 내용
