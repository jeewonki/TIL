레이아웃 2

position
  1. position:static;
    포지션의 기본값, 포지션을 지정해 주지 않았을 때의 원래 위치
  2. position:relative;
    static을 기준으로 위치 지정
    left,top 등 offset 스타일 지정
  3. position:absolute;
    위치가 지정되지 않으면 static 이 기본값
    부모 태그 중 가장 먼저 포지션이 지정된 부모 태그를 기준으로 지정
  4. position:fixed;
    스크롤과는 무관하게 자신의 화면에 고정
    고정된 형태의 UI를 만들 수 있음
flex
list처럼, container의 역할을 할 클래스와 item의 역할을 할 클래스 지정
container에 display:flex;적용 flex-direction:row/column;으로 가로/세로 지정
