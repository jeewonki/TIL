자바스크립트를 넣는 3가지 방법

1. <script> 태그
document.write(): html 문서에 표기
2. 이벤트
태그의 속성으로 자바스크립트가 들어갈 수 있음(onclick처럼 on이 들어가는 속성)
3. 콘솔
웹브라우저 개발자 도구에서 자바스크립트를 한 줄씩 실행 가능함

CSS 디자인 팁

1. HTML 코드에는 기본적으로 패딩과 마진이 들어가 있음
따라서 <html>에 스타일을 적용해 여백을 초기화
2. 자바스크립트 이벤트 속성인 onclick을 사용하면 개체 자체에 링크를 부여할 수 있음
onclick="window.open('address')" onclick="location.href='address'"
