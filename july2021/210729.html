2021.07.29

BOM(Browser Object Model)

웹브라우저의 창 또는 프레임을 제어

전역객체 window

모든 객체들은 window 객체에 소속되어 있다
모든 전역변수는 window 객체의 필드
모든 함수는 window 객체의 메소드
window.alert()와 alert()는 같다

사용자와 커뮤니케이션하기

1. alert()
경고창을 띄운다
'확인'을 눌러서 경고창을 닫기 전까지는 다른 화면은 클릭할 수 없다
경고창이 떠 있는 동안에는 다른 스크립트가 실행되지 않으므로 디버그용으로 많이 사용했지만, 현재는 console.log()를 많이 사용한다

2. confirm()
확인/취소가 있는 창을 생성한다.
확인은 true, 취소는 false의 boolean 을 반환한다.

3. prompt()
텍스트 필드가 있는 창을 생성한다.
입력받은 텍스트를 string으로 반환한다.

Location 객체

현재 윈도우의 URL을 알아낸다.
location.tostring()
location.href()
둘 중 하나를 사용한다.
