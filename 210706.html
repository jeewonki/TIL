Today I Learned(21.07.06)

오픈소스아카데미 Git 교과서
(git.jiny.dev)

깃 저장소 생성
$git init
원격 저장소와 연결
$git remote add <remote> <repository>
$git remote -v
$git remote rm <remote>
$git push -u
$git pull
commit과 스테이지 영역의 개념
$git add <file> $git commit $git status $git log
저장소 복제 및 수정
$git clone <repository> <directory>
브랜치 생성 및 병합
$git branch <branch> $git branch -v $git checkout <branch> $git merge <branch>
리셋/리버트
$git reset <commit ID> $git revert <comit ID>

★리셋은 로컬 저장소에서만 해라
