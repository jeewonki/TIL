최신 CSS 기능

Supports

@supports(display:grid){}

@supports not(display:grid){}

어떤 css 속성이 브라우저에서 지원될 때, 혹은 지원되지 않을 때만 선택적으로 실행된다

스크롤 스냅

scroll-snap-type:y mandatory;//부모 태그에 추가
scroll-snap-align:center;//자식 태그에 추가

자식 태그의 위쪽에 스크롤이 붙는다. 모바일 웹에서, 한 쪽씩 넘기는 UI를 구현할 때 사용할 수 있음

is pseudo selector

header button,
nav button,
form button{
}

:is(header, nav, form) button{
}

여러 개의 선택자를 선택할 때, 공통되는 부분을 묶을 수 있다.

position:sticky

position:sticky

유저 스크롤을 따라다니는 헤더를 만들 수 있다
원하는 요소에 position:sticky 속성을 추가하고,
정의된 높이가 있는 컨테이너 앞에 그 요소를 넣어주면 된다

aspect-ratio

aspect-ratio:16:9;

요소의 가로와 세로 비율을 지정할 수 있다
