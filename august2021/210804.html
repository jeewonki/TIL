Element 객체의 식별자 API

1. Element.tagname

<li id='active'>
var active=document.getElementById('active');
active.tagName='a'//바뀌지 않음

태그네임은 읽기전용 속성이므로, 위 코드와 같이 태그네임을 바꾸어도 문서상으로는 바뀌지 않음

2. Element.id

<li id='active'>
var active=document.getElementById('active');
active.id='deactive';//바뀜

id는 자바스크립트 객체에서 변경 가능

3. Element.className

<li id='active'>
var active=document.getElementById('active');
active.className='a b c';//a, b, c 클래스 추가 active.className='a c';//b 클래스 제거

클래스를 바꾸려면, 클래스의 리스트 전체를 수정해야 함
대신 element,classList를 많이 사용

4. Element.classList

<li id='active'>
var active=document.getElementById('active');
active.classList.add("a");//클래스 추가
active.classList.remove("b");//클래스 제거
active.classList.toggle("c")
//없으면 추가, 있으면 제거

classList 자체는 유사 배열의 형태로, 배열처럼 인덱스를 사용해서 각각의 클래스에 접근 가능

조회 API

document.getElementsBy(""); Element.getElementsBy("");

document에서 호출하면 문서 전체에서, Element에서 호출하면 그 태그의 하위 태그에서 엘리먼트를 추출
