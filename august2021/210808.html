프로퍼티 리스너

(html)
<input type="button" id="mybutton" value="button">
(js)
var button=document.getElementById("mybutton");
button.onclick=function(event){...}
//event.target.value의 값은 'button'

이벤트 대상이 되는 객체의 프로퍼티로 이벤트를 등록
엘리먼트 객체의 프로퍼티에 특정 함수를 적용
event 객체에는 이벤트가 시작된 시점에 대한 여러 가지 정보가 담겨 있음

addEventListerner()

Element.addEventListener('click', function(event){...})

프로퍼티 리스너와는 다르게, 하나의 함수를 여러 오브젝트에 적용, 하나의 대상에 여러 리스너 적용 가능
