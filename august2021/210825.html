외부 자바스크립트 사용

<script src="./source.js"></script>
export class Point{...}
import {Point} from './modules/source.src';

자바스크립트를 별도의 .js 파일로 모듈화할 수 있으며, export, import 문을 사용해 .js 파일 사이에서도 참조가 가능하다.
CORS(Cross Origin Resource Sharing) 은 구글 크롬에서 금지되어 있기 때문에, 도메인이 다른 소스의 교차 사용은 금지된다.
로컬 파일에서는 그 도메인을 특정할 수 없기 때문에, 별도의 방법 없이는 외부 자바스크립트의 사용이 제한된다.
