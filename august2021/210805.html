속성 API

Element.getAttribute(name);
Element.setAttribute(name, value);
Element.hasAttribute(name);
Element.removeAttribute(name);

HTML 태그의 속성(Attribute)를 제어한다
DOM에서 속성에 대응하는 개념에는 프로퍼티가 있으며, 모든 HTML 태그(Element)의 속성은 프로퍼티를 통해 제어할 수 있다.
(tag-Element, Attribute-property)
자바스크립트의 프로퍼티와 HTML의 속성은 다를 수 있다

Event-Driven Programming

Event target: 이벤트가 발생하는 대상(ex)버튼)
Event type: 동작의 종류(ex) 클릭)
Event handler: 실행되는 코드

인라인 방식으로 이벤트 등록

<input type="button onclick=alert("Hello world"+this.value);" value="button">

여기에서 this 객체를 사용하면 자기 참조가 된다.
